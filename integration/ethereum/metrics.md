# Optimum Gateway Metrics Reference

| Metric Name                                                                 | Type         | Labels                                   | How It’s Calculated                                             | Goal                                         |
| --------------------------------------------------------------------------- | ------------ | ---------------------------------------- | --------------------------------------------------------------- | -------------------------------------------- |
| `optp2p_gateway_optimum_gateway_libp2p_total_messages`                      | Counter      | none                                     | Incremented whenever any libp2p message is delivered            | Total throughput of libp2p→gateway messages  |
| `optp2p_gateway_optimum_gateway_libp2p_published_messages_per_topic_total`  | CounterVec   | `topic`                                  | Incremented per-topic in `LibP2PTotalMessagesInc(topic)`        | Track traffic per gossip topic               |
| `optp2p_gateway_optimum_gateway_optimum_total_messages`                     | Counter      | none                                     | Incremented when message is delivered to any Optimum P2P topic  | Total throughput of gateway→Optimum messages |
| `optp2p_gateway_optimum_gateway_optimum_published_messages_per_topic_total` | CounterVec   | `topic`                                  | Incremented per-topic in `OptP2PTotalMessagesInc(topic)`        | Track Optimum P2P topic activity             |
| `optp2p_gateway_optimum_gateway_aggregation_included_total`                 | Counter      | none                                     | Incremented by number of messages batched per flush             | Total batched messages across all topics     |
| `optp2p_gateway_optimum_gateway_aggregation_included_per_topic_total`       | CounterVec   | `topic`                                  | Same as above but per-topic                                     | Measure batching efficiency per topic        |
| `optp2p_gateway_optimum_gateway_libp2p_propagation_latency`                 | GaugeVec     | `topic`                                  | Set to most recent latency (`now - msg.timestamp`)              | Real-time view of last observed latency      |
| `optp2p_gateway_optimum_gateway_libp2p_propagation_latency_milliseconds`    | HistogramVec | `topic`                                  | Each message latency observed into buckets \[1ms–16s]           | Distribution of gossip propagation delays    |
| `optp2p_gateway_optimum_gateway_eth_latency`                                | GaugeVec     | `topic`                                  | Difference between earliest & latest fetch time across gateways | Cross-gateway spread latency (consistency)   |
| `optp2p_gateway_optimum_gateway_message_size_bytes`                         | HistogramVec | `topic`                                  | Each received message size observed into exponential buckets    | Size distribution of messages per topic      |
| `optp2p_gateway_optimum_gateway_message_size_max`                           | GaugeVec     | `topic`                                  | Max size from samples in last 5s                                | Peak message size recently observed          |
| `optp2p_gateway_optimum_gateway_message_size_min`                           | GaugeVec     | `topic`                                  | Min size from samples in last 5s                                | Smallest message size recently observed      |
| `optp2p_gateway_optimum_gateway_message_size_avg`                           | GaugeVec     | `topic`                                  | Mean size from samples in last 5s                               | Rolling average size                         |
| `optp2p_gateway_optimum_gateway_message_size_mean`                          | GaugeVec     | `topic`                                  | Median size from samples in last 5s                             | Typical size (less skewed by outliers)       |
| `optp2p_gateway_optimum_gateway_cl_peers`                                   | Gauge        | none                                     | Set from `Network().Peers()` count of CL libp2p peers           | Monitor CL peer connectivity                 |
| `optp2p_gateway_optimum_gateway_opt_peers`                                  | Gauge        | none                                     | Same for Optimum peers                                          | Monitor Optimum peer connectivity            |
| `optp2p_gateway_optimum_gateway_app_build_info`                             | Gauge        | const labels (`version`, `commit`, `go`) | Set once at startup                                             | Build/version info marker for dashboards     |
